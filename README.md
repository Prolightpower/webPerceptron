# L vs T Perceptron Classifier

A perceptron classifier that predicts whether a drawn symbol is an L or T.

## How it works
- Draw on the 5x5 grid by clicking the cells
- Click Classify to get a prediction
- Uses a perceptron trained on 200 samples (100 L, 100 T)
- Achieves 92.5% test accuracy 

## Files
- index.html - the web app
- style.css - the styling and layout
- script.py - the perceptron predict functions and trained weights 
- lt_data.csv - the training dataset (200 annotated samples)

## How to run
Open the GitHub Pages link in any browser.
